Social Posting Queue — Mon/Wed/Fri 9AM EST
Loading social queue...
Posts fire Mon/Wed/Fri at 9 AM EST via pg_cron → cv-social-poster. Add Instagram and LinkedIn credentials to cv_social_credentials table to activate posting. Image URL must be set per row for Instagram posts.
Configure Social Credentials

Insert credentials into cv_social_credentials table in Supabase:


Instagram: platform=instagram · account_id=[IG Business Account ID] · access_token=[Page Access Token]

LinkedIn: platform=linkedin · account_id=[LinkedIn Person URN] · access_token=[OAuth Token]


Image URLs: set canva_image_url on each queue row after exporting from Canva, or wire the Canva API to auto-export.
